# CoolKids 🎉

**A local family events aggregator for Cherokee County & North Georgia**

CoolKids automatically scrapes local venue websites, finds upcoming family-friendly events, and sends personalized monthly newsletters to parents based on their interests, kids' ages, and how far they're willing to drive.

## Tech Stack

- **Next.js** — React web framework
- **Supabase** — PostgreSQL database
- **Tailwind CSS** — Styling
- **Firecrawl + Apify** — Web scraping
- **Resend** — Email newsletters
- **Vercel** — Hosting

## Getting Started

1. Clone this repo
2. Copy `.env.local.example` to `.env.local` and fill in your keys
3. Run `npm install`
4. Run `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000)
